3-Stage Pipeline

Text first. Vision last.

Tasks flow through the cheapest path. Works with any OpenAI-compatible model.

⚡

Stage 1: SnapshotBuilder

FREE

Parallel OCR + accessibility capture. Produces text elements, buttons, inputs with coordinates. Spatial layout analysis detects zones. No vision needed.

🧠

Stage 2: TextNavigator

CHEAP ($0.25/1M with Haiku)

Primary stage. Text LLM reads OCR snapshot and app guide, decides action. One action per call, verified by re-scan. Handles 90% of tasks.

🎯

Stage 3: VisionFiller

EXPENSIVE, last resort

Fires for CAPTCHAs, complex spatial tasks. Supports Anthropic Computer Use and any vision model.

🔍

Pre-processor

Regex decomposes compound tasks. Zero-cost classifier routes: mechanical (free), navigation (Ctrl+L), reasoning (Stage 2), spatial (vision).

🗺️

Spatial Layout Analysis

Builds text-based spatial map from element clustering. Any text model knows where to click, no screenshots needed.
